AvatarPlus Einstellungen

In diesem Abschnitt gibt es zwei Optionen. Um ein zusätzliches Feld im Kommentarbereich für die sozialen Netzwerke anzuzeigen, haken Sie die Auswahlbox Extra Feld an. Dadurch wird ein zusätzliches Feld im Kommentarbereich angezeigt in dem die Besucher einen Link zu ihren Profil in den sozialen Netzwerken von Google+, Facebook oder Twitter einfügen können.
In einigen Fällen ist es jedoch ratsamer lediglich das Feld für die Homepage, welches ein Standardfeld im Kommentarbereich ihrer WordPress Installation ist, anzuzeigen, da so der Besucher nicht verwirrt wird wo er welche Angaben machen kann bzw. muss. AvatarPlus wird den Link im Feld Homepage auswerten und gegebenenfalls das Profilbild aus dem sozialen Netzwerk anzeigen.
Ist kein Link zu einem Profil angegeben, unabhängig davon ob nun ein zusätzliches oder das Homepage-URL Feld genutzt wird, verwendet AvatarPlus den von WordPress über die E-Mail Adresse erzeugten Avatar. Also entweder den des Gravatar-Services, sofern der User dort mit der eingegebenen E-Mail-Adresse registriert ist, oder den voreingestellten Avatar Ihrer Installation.

AvatarPlus ist auch in der Lage Kurz-URLs von Short-URL Services wie bit.ly oder goo.gl aufzulösen, wodurch die Möglichkeit besteht, dass ein Besucher auch eine URL wie z.B. http://www.example.com/+ als Kurz-URL zu einem Google+ Profil zu verwenden.

Um die Belastung Ihrer Installation bei der Anzeige der Kommentare möglichst gering zu halten, verwendet AvatarPlus einen einfachen Caching-Mechanismus (Cache = Zwischenspeicher). Dabei werden die URLs zu den Profilbildern gespeichert, nicht jedoch die Profilbilder selber. Obwohl dies eine sehr schlanke und effiziente Methode ist Daten temporär zu speichern, kann es sinnvoll sein von Zeit zu Zeit den Zwischenspeicher zu leeren.
So ist es z.B. nicht sinnvoll AvatarPlus-Daten von sehr alten Beiträgen, die nur noch selten oder gar nicht mehr aufgerufen werden, zu speichern. Diese Daten belegen unnötig Platz in der Datenbank und vergrößern somit z.B. ein Datenbank-Backup.
AvatarPlus bietet deshalb die Möglichkeit zwischengespeicherte Daten nach einen gewissen Zeitraum automatisch zu löschen. Tragen Sie hierzu im Auswahlfeld einen Wert ein und wählen dann einen Zeitraum (Tag(e), Woche(n), Monat(e)) aus. Sollen die zwischengespeicherten Daten nicht gelöscht werden, so tragen Sie bitte als Wert eine Null (0) ein. Der ausgewählte Zeitraum ist dann unerheblich.
Bitte beachten Sie, dass in manchen Ländern aus rechtlichen Gründen zwischengespeicherten Daten nach einem festgesetzten Zeitraum zwingend gelöscht werden müssen. Ebenso muss der Besucher Ihrer Seite in manchen Ländern darauf hingewiesen werden, dass zusätzliche, persönliche Daten über ihn gespeichert werden. Sollte diese Rechtslage auf ihren Blog zutreffen, dann geben sie bitte als Art der Daten z.B.: URL zum Profilbild des angegebenen sozialen Netzwerks ein, sowie als Zeitraum für die Dauer der Zwischenspeicherung den von ihnen gewählten Zeitraum (z.B.: 3 Monate) an.

Google+ Einstellungen

Sie benötigen einen API-Schlüssel von Google, falls Sie Profilbilder von Google+ verwenden möchten. Hier (Artikel in englischer Sprache) bekommen Sie nähere Informationen darüber, wie Sie einen solchen API-Schlüssel erhalten.
Nachdem Sie einen API-Schlüssel erhalten haben, tragen Sie ihn bitte in das Eingabefeld für den Google+ API-Schlüssel ein.
